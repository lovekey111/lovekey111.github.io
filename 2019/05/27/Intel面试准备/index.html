Intel面试准备

1、面试Intel优势

  • My major is traffic control engineering which is belong to automatic department. So I have knowledge in software development and algrithm which is the basis of test engineer.
  • I have knowledge in programming languages such as Python, Java, Sql. And I can use them to handle some practical problems.
  • I have been join in some project. And I can finish my job and propose some advice.

2、对项目阐述

(1) casco

  • We obained the data of historical railway operation. It’s not convenient for us to proccess because the data is ordered by time, so we should reform the data. There are also some useless data and we need to drop it.
  • We use the processed data to do some anaylsis, such as statistical analysis and cluster analysis. After that, we can have a overlook of data, and we can find the distribution characteristics of data.
  • We use the data to do some classifications and predictions. Find the different kinds of operation modes and predict the operation delays.

(2) Kaggle

  • Do some exploratory data analysis based on the python libs.
  • Establish a classification model by using Xgboost algrithm.
  • Get the final result by averaging several results.

(3) Tianchi(same as kaggle)

  • The aim of this competition is to perdict the level of happiness. There are about five levels and the larger value the happier.
  • Solve the problem of data imbalance by using oversimpling.

3、遇到最大困难

As the saying goes, everything is difficult at the beginning. It’s difficult to do the research and determine direction of this project.

4、优缺点

  • Advantages: 自我介绍中
  • Disadvantages: work inefficient、I don’t have enough experience and knowledge, sometimes my self-confidence is insufficient.

5、为什么想来英特尔实习

  • 首先是因为英特尔在行业中名气较大，希望能在这样的企业有较好发展。

  • 其次是公司环境好、人才培养方案佳以及团队都较为年轻。充满活力

  • 最后是自己所学知识能够在此发挥作用。

    6、专业不对口，为什么申请这个岗位

  • 我对所学专业认真对待，并取得较好成绩。在学习与实践过程中对编程与算法产生兴趣，并希望从事相关工作。

  • 专业仅仅是知识背景，对于勤于思考、严谨的态度等思维方式，不同专业都是相通的。
